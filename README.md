# awssh

CLI tool to login ec2 instance.

## Install

- Brew

```
$ brew tap youyo/tap
$ brew install awssh
```

Other platforms are download from github release page.

## Requirements

- port forwarding with ssm-agent must be possible
- ec2-instance-connect must be possible

## Usage

```
$ awssh
```
